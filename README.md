# Scrappa
---

## 1. アプリコンセプト

### 1.1 キャッチコピー

**「切り取って、貼って、私をつくる。」**

---

### 1.2 コアコンセプト

**「日常の断片を切り取り、貼り重ねる、デジタルスクラップブック。」**

かつて雑誌の切り抜きをノートに貼っていたように、スマホで撮った日常の断片を切り取ってブックに貼っていく。気になったものを集め続けるうちに、それが記憶の蓄積になり、自分だけのコレクションになる。

---

### 1.3 世界観

**アナログの温かさ × デジタルの便利さ**

| アナログの要素 | デジタルの要素 |
|---|---|
| リングノートの質感 | スマホで即保存 |
| 切り抜きを貼る感覚 | 正方形トリミング |
| ページをめくる体験 | どこからでも見返せる |
| 手帳に育てていく | フレンドとゆるく共有 |

---

### 1.4 Scrappaが解決する課題


| 悩み | 現在の代用ツール | Scrappaの解決 |
|---|---|---|
| 撮った写真が埋もれる | カメラロール | ブックとして積み重なる・見返せる |
| 整理が続かない | Notion | 整理しなくていい・貼るだけ |
| 作り込みが大変 | GoodNotes | 正方形クロップだけ・シンプル |
| 発信したくないものが埋もれる | Instagram保存 | 自分のためだけのスペース |

## 2. 提供する体験

### 2.1 体験の流れ

```
① 出会い
  街で気になるものを見かける
  「なんかいいな」「これ好きかも」
  ↓
② 切り取り
  撮影して、好きな部分だけトリミング
  ↓
③ 積み重なり
  ブックにクリップが増えていく
  意識せずに続けているうちにページが埋まっていく
  ↓
④ 発見
  ふと見返したとき、クリップ同士の
  共通点・パターン・関係性に気づく
  「自分、こういうの好きだったんだ」
  ↓
⑤ 記憶の蘇り
  あのとき、どこで、なぜ気になったのか
  クリップが記憶のトリガーになる
```

---

### 2.2 体験の3本柱

#### 体験① 記憶の蓄積
写真全体ではなく「気になった部分だけ」を切り取るから、記憶の純度が高い。ページをめくるたびに、あのとき・あの場所・あの感覚が戻ってくる。写真日記とは異なり、全体を記録しようとしなくていい。断片だからこそ、記憶の純度が高い。

#### 体験② 自分のコレクション
「赤いもの」「丸いロゴ」「和な感じのパッケージ」など、自分でテーマを決めて収集できる。集まっていくほどに達成感があり、ブックが育っていく感覚がある。コレクターとしての楽しさと、自分の感性の可視化が同時に起きる。

#### 体験③ 偶発的な発見
能動的に「記録しよう」とせず、気になったから貼っただけなのに、後から意味が見えてくる。積み重なったクリップ同士に関係性・共通点・パターンが浮かび上がる。その偶発性こそがScrappaの醍醐味。

---

### 2.3 断片性という独自価値

Scrappaが最もユニークな点は**「断片を切り取る」**という行為が設計の中心にあること。

- 写真日記 → 全体を記録する
- カメラロール → 全体が保存される
- **Scrappa → 気になった部分だけを切り取る**

断片だからこそ、記憶の純度が高く、見返したときの引っかかりが強い。

## 3. ターゲットユーザー

### 3.1 メインターゲット

**「日常をちゃんと残したい、収集が好きな18〜35歳」**

| 軸 | 詳細 |
|---|---|
| 行動 | 気になるものをとりあえず撮る・好きなものを集める癖がある |

### 3.2 サブターゲット

お菓子・飲料・コスメのパッケージを集める層。すでに収集習慣があるため、デジタル版の受け皿として自然にはまる。

## 4. 競合との差別化

### 4.1 ポジショニングマップ

```
                    他人のコンテンツが主役
                           │
              Pinterest     │    Cosmos
              Instagram保存 │    （クリエイター向け）
                           │
自分で作らない ─────────────┼───────────────── 自分で作る
（保存・ブックマーク）      │          （デコ・レイアウト）
                           │
                           │  Picrap    GoodNotes
                           │  （思い出アルバム）
                    自分のコンテンツが主役
                           │
                       Scrappa ●
                    （日常の断片・収集）
```

Scrappaは「自分のコンテンツが主役」×「作ること自体はシンプル」というポジションで、どの競合とも重ならない

---

### 4.2 主要競合との比較


競合と比較したとき、Scrappaにしかない要素は3つ。

**① 自分で撮った断片が主役**
他のどのサービスも「自分が撮影した写真の断片」を主役に置いていない。

**② 正方形トリミングによる統一感**
切り取って貼るという行為がアプリの設計に組み込まれている。断片を切り取ることで記憶の純度が上がる。

**③ ノートという器の体験**
ページをめくる・積み重なるという物理的なスクラップブックの感覚をデジタルで再現している。
